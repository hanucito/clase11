# GraphQL Connections. Intro Relay

- Introduccion a GraphQL connections

## Correr servidor localmente

```
cd graphql-server
```
```
npm intall
```
```
DBUSER=xxxxxx DBPSW=xxxxxx npm start
```

```
cd relay-modern-app
```
```
npm intall
```
```
npm start
```
